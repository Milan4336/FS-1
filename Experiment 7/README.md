# Experiment 7: Full Stack React Hub

A state-of-the-art Full Stack application demonstrating the integration of React with an Express backend, featuring real-time communication, global state management, and asynchronous data fetching.

## 🚀 Overview

This experiment is divided into three key sub-experiments, each focusing on a critical aspect of modern web development:

1.  **Sub 7.1: Axios & API Integration**: Demonstrates how to connect a React frontend to an Express REST API using Axios, including robust loading and error state management.
2.  **Sub 7.2: Redux Toolkit Cart**: Implements complex local state management using Redux Toolkit. Features include add/remove logic, quantity updates, and persistent storage via `localStorage`.
3.  **Sub 7.3: Real-Time Chat (Socket.io)**: A live communication hub with bidirectional messaging, typing indicators, and a dynamic online user counter.

## 🛠️ Technology Stack

-   **Frontend**: React (Vite), Redux Toolkit, Axios, Socket.io-client, Bootstrap, React-Bootstrap, Lucide React.
-   **Backend**: Node.js, Express, Socket.io, CORS, Dotenv.
-   **Design**: Signature Indigo-Pink Glassmorphism aesthetic.

## 📂 Project Structure

-   `server/`: Contains the Express server, REST endpoints, and Socket.IO configuration.
-   `client/`: Contains the React application, Redux store, and UI components.

## 🔧 Installation & Setup

1.  **Backend**:
    -   Navigate to `/server`
    -   `npm install`
    -   `node server.js` (Runs on Port 3007)

2.  **Frontend**:
    -   Navigate to `/client`
    -   `npm install`
    -   `npm run dev` (Runs on Port 5173/5174)

## ✨ UI Features

-   **Drifting Blobs**: Signature animated background for visual continuity.
-   **Glassmorphism**: High-blur filters and semi-transparent cards.
-   **Responsive Design**: Built with Bootstrap for multi-device support.
